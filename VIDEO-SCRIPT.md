# flightrec-slack — demo video script (2:57, New Slack Agent track)

Goal: in under 3 minutes, hit all four EQUAL judging criteria — Quality of Idea,
Technological Implementation, Potential Impact, Design.

Structure: **build up, then break.** The video earns the tamper reveal instead of
cold-opening on it — problem, product, happy path, the gate holding against an
over-reach, a verified audit… and only then the question ("what if someone
rewrites the history?"), the attack, and the INTEGRITY VIOLATION flip at 2:10 as
the climax. The MCP beat and close ride the payoff.

## How the video is made (Remotion)

The video is fully generated — no screen recording. Source: `video/remotion/`.

```
cd video/remotion
npm install          # first time only
npm run render       # syncs artifacts + renders ../flightrec-demo.mp4 (1080p30)
npm run preview      # Remotion Studio, scrub the timeline interactively
```

- Every beat, caption, and chip hangs off ONE table: `src/timeline.ts`.
  Re-timing the video is a one-file edit.
- All on-screen data (run ids, hashes, rationales, token counts) imports from
  `video/artifacts.js` — REAL pipeline output generated by
  `video/gen_artifacts.mjs` (live Claude reasoner + real tamper) and
  `mcp/scripts/gen_mcp.mjs` (real MCP stdio round-trip). Never hand-edit it.
- Spot-check a frame: `npx remotion still src/index.ts flightrec-demo out.png --frame=N`
- Legacy: `video/demo.html` + `video/record.mjs` (the Playwright screencast
  toolkit that produced v1) are superseded by `video/remotion/` and kept only
  for reference.

## Criteria the judges are scoring (named on screen as chips)

- **Quality of Idea** — a tamper-evident black box for AI agents; doesn't exist as a Slack agent.
- **Technological Implementation** — MCP server integration (the track's qualifying technology) + SHA-256 hash chain + append-only SQLite + 26 real-DB tests.
- **Potential Impact** — unlocks AI agents for regulated orgs; AI governance is the #1 enterprise-AI blocker.
- **Design** — verdict-header cards, two-column field grids, one-click audit, the earned FAIL card.
